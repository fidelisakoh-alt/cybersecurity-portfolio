# Google Sites — Portfolio Outline (Copy/Paste Friendly)

## Pages
1. **Home**
   - Photo + short professional statement (3–4 lines)
   - Buttons: View GitHub • Download CV (Google Drive link)

2. **Projects**
   - Cards for 4–6 strongest projects
   - Each card: title, 2–3 bullet outcomes, "View case study" (links to GitHub file)

3. **Incident Journal**
   - Brief explanation + selected anonymised entries
   - Link: "See full journal on GitHub"

4. **Labs**
   - Linux permissions, SQL, Wireshark/SIEM summaries
   - Link to relevant repositories/folders

5. **About**
   - Why cybersecurity, your strengths, tools you use
   - Contact (professional email / LinkedIn)

## Style Tips
- Keep it simple, readable, and responsive by default.
- Use screenshots sparingly; always redact.

## Embeds
- GitHub file embed: paste the GitHub URL (or link out).
- Google Drive: set file to “Anyone with the link: Viewer” and embed.
